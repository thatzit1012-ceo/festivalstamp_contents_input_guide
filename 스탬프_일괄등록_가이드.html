1 준비물
  • 엑셀 파일 (.xlsx) — 스탬프 정보 목록 (관리자 페이지에서 양식 다운로드)
  • 이미지 파일 — 각 스탬프의 대표 이미지 (없어도 됩니다 — 기본 이미지 자동 적용)
⚠️ 이미지를 제공하지 않은 스탬프에는 '이미지 준비중' 기본 이미지가 자동으로 노출됩니다.
2 엑셀 파일 작성 방법

아래 6개 컬럼을 사용합니다. 관리자 페이지에서 양식을 다운로드하면 헤더가 미리 준비되어 있습니다.

순번 필수

1부터 순서대로 입력합니다.
이미지 파일명과 반드시 일치해야 합니다.

예) 1, 2, 3, 4 …
스탬프명 필수

스탬프(장소/콘텐츠)의 이름입니다.
앱 화면에 표시되는 이름입니다.

예) 화성 효 마라톤 대회
설명 선택

스탬프에 대한 간단한 부연 설명입니다.

예) 매년 5월 개최되는 마라톤 행사
주소 선택

장소의 도로명 또는 지번 주소입니다.

예) 경기도 화성시 반송동 1
위도 선택

지도 표시용 위도입니다.
소수점 4자리 이상 권장합니다.

예) 37.1990
경도 선택

지도 표시용 경도입니다.
소수점 4자리 이상 권장합니다.

예) 126.8320
✏️ 작성 시 주의사항
• 순번은 1부터 연속된 숫자로 입력 — 이미지 파일명(1.png, 2.jpg …)과 반드시 일치
• 스탬프명이 비어있는 행은 자동으로 건너뜁니다
• 위도·경도는 소수점 형식으로 입력 (예: 37.5665 / 126.9780)
• 행 순서가 앱에서 스탬프 목록의 표시 순서가 됩니다
3 이미지 파일 준비 방법

이미지 파일명을 순번.확장자 형식으로 지정하면 자동으로 해당 스탬프와 매칭됩니다.

규칙내용
파일명 형식순번.확장자   예) 1.png, 2.jpg, 3.jpeg
지원 형식png  ·  jpg  ·  jpeg  ·  gif  ·  webp
저장 위치ZIP 안의 images/ 폴더
이미지 없을 때'이미지 준비중' 기본 이미지 자동 적용
권장 해상도800 × 800 픽셀 이상 (정사각형 권장)
이미지 매칭 예시
순번 1→images/1.png1번 스탬프 이미지로 등록
순번 2→images/2.jpg2번 스탬프 이미지로 등록
순번 3→(파일 없음)기본 이미지 '이미지 준비중' 자동 적용
순번 4→images/4.jpeg4번 스탬프 이미지로 등록
4 ZIP 파일 구성 방법

엑셀 파일과 이미지 폴더를 하나의 ZIP 파일로 묶습니다.

ZIP 내부 구조
upload.zip
├── 스탬프목록.xlsx    ← 엑셀 파일 (파일명 자유)
└── images/            ← 폴더명은 반드시 'images'
    ├── 1.png
    ├── 2.jpg
    ├── 3.png
    └── 4.jpeg
ZIP 만드는 방법 (Windows)
  • 엑셀 파일과 images 폴더를 같은 위치에 놓습니다
  • 두 항목을 함께 선택 (Ctrl+클릭)
  • 마우스 우클릭 → '압축(ZIP 폴더로)' 또는 '보내기 → 압축(ZIP) 폴더'
  • 생성된 .zip 파일을 업로드에 사용합니다
⚠️ images 폴더명은 반드시 영문 소문자 images 로 지정해야 합니다.
⚠️ 엑셀 파일은 ZIP 최상위에 위치해야 합니다 (하위 폴더 안에 있으면 인식 안 됨).
5 자주 묻는 질문 (FAQ)
Q1. 이미지 없이 엑셀만 업로드해도 되나요?
네, 가능합니다. images/ 폴더 없이 엑셀만 ZIP에 포함하거나, 이미지가 없는 스탬프는 파일을 생략하면 '이미지 준비중' 기본 이미지가 자동 적용됩니다.
Q2. 이미 스탬프가 등록된 행사에 추가 업로드해도 되나요?
스탬프 콘텐츠 추가는 가능합니다. 단, 추가된 항목은 기존 항목들 뒤에 위치합니다. 기존 항목들 사이에 위치하는 것은 불가합니다.
Q3. 업로드 중 일부만 실패하면 어떻게 되나요?
성공한 항목은 정상 등록되고, 실패한 항목만 결과 화면에 오류 내역이 표시됩니다. 실패 항목만 수정하여 별도 ZIP으로 재업로드하면 됩니다.
Q4. ZIP 파일 크기 제한이 있나요?
100MB 이하를 권장합니다. 이미지가 많을 경우 800px 내외로 리사이즈하여 용량을 줄여주세요.
Q5. 이미지 형식이 JPG인데 PNG로 바꿔야 하나요?
아니요, png · jpg · jpeg · gif · webp 모두 지원합니다. 파일명만 순번.jpg 형식으로 맞춰주면 됩니다.
